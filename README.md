# La-Torta bot — исправленная загрузка каталога

Эта версия не зависит от sitemap и JSON-LD карточек.

## Как работает каталог

1. Бот открывает реальную HTML-карту сайта:
   `https://la-torta.ua/ua/siteindex/`
2. Из неё получает категории La-Torta.
3. Открывает категории как браузер Chrome через `curl_cffi`.
4. Сохраняет товары непосредственно из листинга категории:
   название, цену, фото и реальную ссылку.
5. Обходит страницы пагинации.
6. AI не имеет права создавать ссылки.

## Развёртывание

Замените файлы в GitHub и выполните в Render:

`Manual Deploy` → `Clear build cache & deploy`

Это важно, потому что добавлена новая зависимость `curl_cffi`.

## Проверка

После запуска:

1. `/refresh`
2. Через несколько минут `/status`
3. `/searchtest шоколад`
4. `/searchtest барвник`

Нормальный `/status` должен показывать:

- последний HTTP: 200;
- категорий найдено: больше 100;
- товаров в базе: больше 0.

Если последний HTTP не 200, пришлите полный ответ `/status`.
